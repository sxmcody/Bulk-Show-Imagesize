### python练习用

网上看到有人需要批量显示图片的物理尺寸（厘米或毫米为单位，而非px），所以作为python练习就尝试着写了这个脚本。

##### 脚本功能

遍历指定文件目录，剔除非图片文件，再获取所有的图片像素尺寸以及dpi，通过公式转换获得物理尺寸（厘米为单位）。
